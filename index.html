➕ 새 프로젝트 만들기

📖 Sprint Manager 사용자 매뉴얼

목차 : 1. 소개 2. 화면 구성 3. 프로젝트 관리 4. 스프린트 표 5. 타임라인 6. 월별 달력 7. 스프린트 메모 8. 엑셀 9. 실행 취소 10. FAQ

1. 이 페이지는 무엇인가요?

Sprint Manager는 개발 프로젝트의 스프린트(Sprint) 일정을 한눈에 관리하는 도구입니다. 하나의 스프린트는 보통 세 단계로 구성됩니다.

단계색상의미
Dev🟦 파랑개발 기간
Sanity🟧 주황기획 검증 기간
QA🟩 청록품질 검증 기간
⚠️ 중요: 이 페이지는 브라우저에서만 동작하며 새로고침하면 데이터가 사라집니다. 작업 후 반드시 엑셀 내보내기로 백업하고, 다음에 불러오기로 복원하세요.

2. 화면 구성

  • 왼쪽 어두운 영역(사이드바) — 프로젝트 목록
  • 오른쪽 메인 영역 — 위에서부터: 프로젝트 제목 → Excel 툴바 → 범례 → 스프린트 요약 표 → 타임라인 → 월별 달력

3. 프로젝트 관리 (사이드바)

3-1. 프로젝트 전환

사이드바에서 프로젝트 이름을 클릭하면 메인 화면이 즉시 전환됩니다. 현재 프로젝트는 파란 배경으로 표시됩니다.

3-2. 새 프로젝트 만들기

  1. 사이드바 하단 ［＋ 새 프로젝트］ 클릭
  2. 이름 입력 + 12가지 색상 중 선택
  3. ［만들기］ 클릭 → 빈 프로젝트가 생성되고 자동 전환됨

3-3. 이름 변경 / 삭제 / 사이드바 접기

  • 이름 변경: 메인 화면 상단 프로젝트 제목 클릭 → 수정 → Enter
  • 삭제: 사이드바 프로젝트에 마우스 올림 → × 클릭 (마지막 1개는 삭제 불가)
  • 접기: 사이드바 상단 ☰ 클릭

4. 전체 스프린트 요약 (표)

표가 모든 데이터의 중심입니다. 여기서 수정하면 타임라인·달력에 자동 반영됩니다.

4-1. 날짜 수정

날짜 칸을 클릭 → 달력에서 선택. 즉시 영업일 수가 자동 재계산(주말 제외)되고 화면 전체가 갱신됩니다.

💡 비고에 광복절(08/15) → QA 처럼 공휴일명(단계) 형식으로 쓰면 해당 단계 영업일에서 공휴일만큼 자동 차감됩니다.

4-2. 스프린트 추가 / 삭제 / 순서 변경

  • 추가: 표 위 파란 ［＋ 스프린트 추가］ 버튼. Dev 시작일은 직전 QA 종료일의 다음 월요일로 자동 설정.
  • 삭제: 행의 × 버튼 → 확인. 번호 자동 재정렬.
  • 순서 변경: 행 왼쪽 ⠿ 핸들을 누른 채 드래그 → 파란 줄 위치에 놓기. 번호와 메모가 새 순서로 자동 재부여.

5. Sprint 타임라인 (간트)

  • 1칸 = 1주, 좌우 스크롤 가능. 범위는 스프린트 날짜에 맞춰 자동 조절.
  • 단계 행: Dev/Sanity/QA 바 표시. Sprint 행: 전체 기간 바 표시.
  • 이벤트 행: 칸 클릭 → 이벤트명 입력 + 색상 선택 → 저장. 지우려면 다시 클릭 → 🗑 지우기.

6. 월별 달력

  • 배경색 = 스프린트 기간(홀수 주황/짝수 파랑), 가는 바 = 진행 중인 단계, 태그 = 단계 시작일
  • 날짜에 마우스 올리면 툴팁으로 스프린트·단계 확인
  • 왼쪽 열 숫자(예: 29W)는 연간 주차, 빨강=일요일·공휴일, 파랑=토요일
  • 월별 메모: 달력 카드의 📝 버튼 → 마크다운 입력(/로 포맷 메뉴) → 미리보기 → 저장

7. 스프린트 메모

  1. 표에서 스프린트 번호 옆 📝 버튼 클릭
  2. 마크다운으로 작성, /로 포맷 메뉴 사용 가능
  3. 상단 ‹ › 버튼으로 이전/다음 스프린트 메모로 이동 (작성 내용 자동 저장)
  4. 메모가 있으면 📝 버튼이 노란색으로 표시

8. 엑셀 내보내기 / 불러오기

8-1. 내보내기 (백업)

［⬇ 내보내기］ 클릭 → 프로젝트명_날짜_시분초.xlsx 다운로드. 시트 4개: 스프린트일정 / 월별메모 / 스프린트메모 / 이벤트.

8-2. 엑셀에서 수정 시 규칙

  • 날짜는 YYYY-MM-DD 형식 (예: 2026-07-13)
  • 스프린트 열은 S11처럼 S+숫자 형식
  • 시트 이름은 변경 금지

8-3. 불러오기

［⬆ 불러오기］ → 파일 선택 → 표·타임라인·달력 자동 갱신. 현재 선택된 프로젝트에 적용되므로 다른 프로젝트에 넣으려면 먼저 전환하세요.

9. 실행 취소 / 다시 실행

단축키동작
Ctrl + Z (Mac: ⌘+Z)실행 취소
Ctrl + Shift + Z 또는 Ctrl + Y다시 실행

최대 50단계까지 가능하며 날짜·스프린트·메모·이벤트·프로젝트의 모든 변경이 대상입니다.

10. 자주 묻는 질문 (FAQ)

Q. 새로고침했더니 데이터가 사라졌어요.
A. 데이터는 서버에 저장되지 않습니다. 작업 후 엑셀 내보내기로 백업하고 불러오기로 복원하세요.

Q. 영업일 수를 직접 고치고 싶어요.
A. 날짜와 비고의 공휴일 정보로 자동 계산되는 값이라 직접 수정할 수 없습니다.

Q. 달력에 중간 월이 안 보여요.
A. 스프린트의 전체 기간(시작~종료)이 설정되어 있으면 사이 모든 월이 자동 표시됩니다. 전체 기간 칸을 확인하세요.

Q. 한 프로젝트 일정을 다른 프로젝트로 복사하려면?
A. 원본에서 내보내기 → 대상 프로젝트로 전환 → 불러오기.
